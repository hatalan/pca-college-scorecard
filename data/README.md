# Data

This directory is gitignored. To reproduce the analysis, download the following files into this directory.

## College Scorecard Data

From the [U.S. Department of Education College Scorecard](https://collegescorecard.ed.gov/data):

- **Most-Recent-Cohorts-Institution.csv** — Download "Most Recent Institution-Level Data" from the College Scorecard data page.
- **CollegeScorecardDataDictionary.xlsx** — Available on the same page under documentation/data dictionary.
